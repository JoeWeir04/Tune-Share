# Tune Share - a music sharing web app

Tune Share is a music sharing web application that allows users to upload music videos, like posts, and comment on them.  
Built utilizing the **Django** python framework, using **JS**, **CSS** and **HTML**. The Bootstrap library was used to style the web app giving a uniform aesthetic. AJAX was used for live updates of posts likes and comments. Implemented accounts enabling registering, logins and user profile customization such as user profile picture.

## How to Run

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
1. Populate Database:
   ```bash
   python populate_MVapp.py
1. Run the server:
   ```bash
   python manage.py runserver
